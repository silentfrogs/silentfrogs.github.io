Welcome to Science and Tech CCA Trials

What you need to do for the CCA Trials

1

Decide 1-2 Sections of the CCA you wish to do a test for

2

Open the tests on the navigation bar above (press the icon with three lines)

3

Brief Description of the Various CCA Sections

Programming

Code the Robots to do actions to complete competition objectives. Code programs to solve problems

Building

Build robots for various competitions. Build attachments to solve missions

Presentation

Present solutions to problems as well as present ideas.

Science

Do science experiments and learn about science beyond the syllabus
